# 실버스텝 현장 테스트 가이드 (4주차)

> 작성일: 2026-07-20 | 대상: 시니어 3~5명 방문 테스트

---

## 📦 방문 전 준비물 체크리스트

### 필수 준비물
- [ ] **안심 위치 매트** — 다이소 원형 거실 매트 또는 발바닥 스티커 (밝은 초록색 권장)
- [ ] **스마트폰 거치대** — 3m 거리에 안정적으로 고정 가능한 거치대
- [ ] **보조배터리/충전기** — 테스트 중 배터리 부족 방지
- [ ] **관찰자용 기기** — 필드 테스트 대시보드 접속용 (태블릿 또는 노트북)
- [ ] **동의서** — 테스트 참여 동의서 (사진/영상 촬영 여부 포함)
- [ ] **사전 세팅 URL** — 어르신별 field-setup.html 링크 준비

### 선택 준비물
- [ ] 여분 스마트폰 (1대 더)
- [ ] 블루투스 스피커 (음성 증폭용)
- [ ] 간식/음료 (어르신 대접용)
- [ ] 명함 또는 연락처 카드

## 🔧 어르신별 세팅 절차

### 1단계: 사전 URL 생성 (방문 전)

각 어르신에 대해 미리 세팅 URL을 준비합니다:

```
https://[호스팅주소]/field-setup.html?senior=홍길동&child=홍영희&webhook=https://hook.make.com/xxx&target=5
```

| 파라미터 | 설명 | 예시 |
|----------|------|------|
| `senior` | 어르신 이름 | 홍길동 |
| `child` | 자녀 이름 | 홍영희 |
| `webhook` | Make 웹훅 URL | https://hook.make.com/xxx |
| `target` | 목표 횟수 | 5 (첫 테스트는 5회 권장) |

### 2단계: 현장 세팅 (방문 시)

1. **스마트폰 위치 결정**
   - 거실 한쪽 벽면 또는 TV 옆 3m 거리 확보
   - 스마트폰을 거치대에 고정, 전면 카메라가 거실 중앙을 향하도록

2. **안심 위치 매트 배치**
   - 스마트폰으로부터 정확히 3m 거리에 매트 배치
   - 카메라 화각 중앙에 매트가 위치하도록 조정

3. **field-setup.html 실행**
   - 어르신 스마트폰 브라우저에서 사전 준비한 URL 열기
   - 위저드 5단계를 순서대로 진행
   - 특히 카메라/음성 테스트를 3m 거리에서 반드시 확인

4. **PWA 홈 화면 추가**
   - Chrome → 메뉴(⋮) → "홈 화면에 추가"
   - 홈 화면에 실버스텝 아이콘이 생성되었는지 확인

## 🔍 Zero-Touch 관찰 프로토콜

### 관찰 원칙

| 원칙 | 설명 |
|------|------|
| **비개입 관찰** | 어르신이 스스로 조작하는 모습을 지켜봅니다 |
| **도움 레벨 기록** | 개입이 필요할 때 도움 수준을 기록합니다 |
| **안전 우선** | 낙상 위험이 보이면 즉시 개입합니다 |

### 도움 레벨 분류

| 레벨 | 설명 | 대시보드 태그 |
|------|------|-------------|
| **L0** | 완전 자립 — 도움 없이 완수 | ✅ 성공 |
| **L1** | 구두 안내 — "여기 서세요" 등 말로만 도움 | 🆘 도움 |
| **L2** | 물리적 도움 — 직접 위치를 잡아드리거나 버튼 터치 | 🆘 도움 + 메모 |
| **L3** | 중단 — 안전/기술 문제로 테스트 중단 | ❌ 실패 |

### 관찰 절차

```
1. 관찰자 기기에서 field-test-dashboard.html 열기
2. "참여자" 탭에서 정보 입력 후 저장
3. "관찰 기록" 탭에서 "관찰 시작" 클릭
4. 어르신에게 "홈 화면의 실버스텝 앱을 열어주세요" 안내
5. 이후 비개입 관찰 시작
6. 이벤트 발생 시 실시간으로 성공/실패/도움/메모 기록
7. 운동 완료 후 "관찰 종료" 클릭
8. "체크리스트" 탭에서 Zero-Touch 체크리스트 체크
```

## ⚠️ 주의사항

1. **안전이 최우선**: 어르신이 비틀거리거나 힘들어하시면 즉시 중단
2. **부담 주지 않기**: "못 하시는 것"이 아니라 "앱의 문제"라고 안내
3. **시간 존중**: 전체 방문 시간은 30분~1시간 이내
4. **사후 감사**: 테스트 후 감사 인사 + 간식/음료 대접
