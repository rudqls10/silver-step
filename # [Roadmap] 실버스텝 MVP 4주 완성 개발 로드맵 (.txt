- **2주차 (7/13 ~ 7/19)**: 노코드 UI 화면 구성 및 카메라-AI 핵심 로직 연결
- **3주차 (7/20 ~ 7/26)**: 카카오톡 알림 연동 및 개발자 E2E 자체 테스트 완료

## 🏃‍♂️ 주차별 상세 액션 플랜

### 1주차: AI 모션 모델링 및 VUI 에셋 빌딩
- [ ] **AI 포즈 데이터 수집**: 스마트폰 카메라를 두고 3m 뒤 거실 환경에서 직접 '서 있는 자세', '앉은 자세(운동 완료)', '이탈 자세' 촬영
- [ ] **Google Teachable Machine 학습**: 수집한 사진/영상을 넣어 포즈 모델(Pose Model) 클래스 3개 생성 및 웹 내보내기 URL 확보
- [ ] **성우 TTS 음성 추출**: Typecast 또는 ElevenLabs를 활용하여 PRD에 정의된 대사 4종을 다정한 성우 목소리의 mp3 파일로 다운로드[cite: 1]

### 2주차: 노코드(No-code) UI 및 비전 로직 구현
- [ ] **프론트엔드 UI 스케치**: 플러터플로우(FlutterFlow) 또는 버블(Bubble)에서 카메라 화면이 중심이 되는 미니멀 페이지 설계
- [ ] **시각 소거 UX 반영**: 어르신의 프라이버시를 위해 카메라 배경은 블러 처리하고 관절 스켈레톤만 가볍게 시각화[cite: 1]
- [ ] **비전-음성 트리거 코딩**:
  - 앱 로드 시 $\rightarrow$ 오디오 1 재생
  - AI 가 안심위치 포즈 인식 시 $\rightarrow$ 3초 카운트다운 후 오디오 2 재생[cite: 1]
  - 포즈 상태가 변할 때마다 카운트 +1 처리

### 3주차: 외부 API 자동화 연동 및 알파 테스트
- [ ] **백엔드 시나리오 설계**: Make (구 Integromat) 플랫폼 가입
- [ ] **카카오톡 알림톡 연동**: 운동 종료 버튼 클릭(또는 카운트 완료) 시 Make 웹훅을 통해 비즈엠 알림톡 API가 실행되도록 세팅
- [ ] **알파 테스트**: 내 스마트폰으로 직접 3m 뒤에 서서 음성이 타이밍에 맞게 나오는지, 카톡이 자녀 폰으로 잘 가는지 E2E 테스트 진행

### 4주차: 리얼 월드 시니어 현장 검증 (오프라인)
- [ ] **방문 코딩/테스트 준비**: 다이소에서 '안심 위치'를 나타낼 수 있는 원형 거실 매트 또는 발바닥 스티커 준비
- [ ] **현장 Zero-Touch 테스트**: 친척 어르신이나 타깃 시니어 3~5명의 집 거실을 직접 방문하여 세팅해 드린 후, 터치 없이 3m 뒤에서 음성만으로 운동을 완수하는지 밀착 관찰
- [ ] **인터뷰 및 정성 피드백 수집**: 자녀에게 알림톡이 갔을 때 자녀의 실제 안부전화 반응이 오는지 확인 후 시장성 최종 검증
